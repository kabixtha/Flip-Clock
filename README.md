
# 🕐 Flip Clock

A beautiful animated flip clock web application built with Django and deployed on Railway.

## 🌐 Live Demo

👉 [flip-clock-production.up.railway.app](https://flip-clock-production.up.railway.app)

## 🛠️ Built With

- **Python** — Backend language
- **Django** — Web framework
- **Gunicorn** — Production web server
- **WhiteNoise** — Static file serving
- **Railway** — Cloud deployment

## 🚀 Run Locally

1. Clone the repo:
   ```bash
   git clone https://github.com/kabixtha/Flip-Clock.git
   cd Flip-Clock
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the server:
   ```bash
   python manage.py runserver
   ```

4. Open in browser:
http://127.0.0.1:8000
